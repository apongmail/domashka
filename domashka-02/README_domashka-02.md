# Домашка №2 — Agile Manifesto vs. мій робочий процес

**Автор:** Андрій Пономарьов / Claude Code Opus 4.8

Рефлексія у п'ять кроків:

1. **Agile Manifesto** — 4 цінності + суть 12 принципів.
2. **Чому еволюціонував SDLC** — особистий погляд (Waterfall → Agile →

4. **Відмінності** від «класичного» agile (порівняльна таблиця).
5. **Покращення** з акцентом на організацію роботи в команді.
